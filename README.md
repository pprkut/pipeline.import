# Pipeline.Import

Pipeline.Import is a module for loading data into a system.

Installation
------------

* Install Pipeline.Import with Composer or with your own installer.
* Pipeline.Import follows the [semantic versioning][2] standards.

Community
---------

* Follow us on [GitHub][3].

Contributing
------------

Pipeline.Import is an Open Source, community-driven project. Join by contributing code or documentation.

If you encounter any issues when using Pipeline.Import you can report them [on github][4]
